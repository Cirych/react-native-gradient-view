# react-native-gradient-view

Installation

```sh
npm install @cirych/react-native-gradient-view
```

Usage

```javascript
import { GradientViewView } from "@cirych/react-native-gradient-view";

// ...

<GradientViewView colors={['red', '#00a852dd', '#ffa852', 'brown', 'blue']} angle={15} />
```
